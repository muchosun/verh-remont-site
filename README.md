# ВЕРХ ремонт: версии прототипа сайта

Статичный сайт для GitHub Pages с версионированием макетов.

## Что внутри

- `/v1/` - первый кликабельный прототип;
- `/v2/` - версия ближе к сгенерированным desktop/mobile макетам;
- `/v3/` - версия с правками по Telegram: материалы по уровням ремонта, каталог обоев, контакты Telegram/MAX/VK/WhatsApp;
- `/v4/` - версия по полному разбору комментариев Юры: преимущества, быстрые контакты, подробные уровни ремонта и расчет от выбранного уровня;
- `/v5/` - сайт-квиз как интерактивная презентация: выборы сразу переводят к следующему шагу, заявка собирается в состоянии квиза, финал просит только телефон;
- `/v6/` - mobile-first квиз на одном экране: фиксированная высота без прыжков, фото-ответы, компактный прогресс и короткая заявка;
- `/v7/` - контентный сайт с квизом в iOS-style поп-апе: страница прогревает, квиз открывается поверх сайта;
- `/v8/` - версия с оригинальными фотографиями из VK: hero, тарифы и фото-выборы квиза заменены на реальные объекты ВЕРХ ремонта;
- `/v9/` - критичная правка мобильной воронки: ценовой якорь на первом экране, липкая CTA только после прокрутки, компактный квиз;
- `/v10/` - расчет показывается только после телефона: квиз собирает контакт, показывает лоудер и открывает ориентир по бюджету; визуалы типов квартир и уровней отделки заменены;
- `/v11/` - галереи уровней отделки: свайп, стрелки, точки и мягкое автолистание только когда блок виден на экране;
- `/v12/` - исправлен ручной ввод площади в квизе: поле можно очистить и набрать свое значение, включая десятичные метры;
- `/v13/` - правки по замечаниям и реальные фото объектов для уровней Стандарт, Комфорт и Люкс;
- `/v14/` - клиентские тексты, каталоги материалов и график платежей 30/30/20/20 после телефона;
- `/v15/` - зафиксирован дизайн-стиль с основным шрифтом Onest, подключенным локально;
- `/v16/` - темная система из UI kit, бело-оранжевый логотип, более легкие начертания интерфейса и CTA после анимированных этапов ремонта;
- `/v17/` - 3D-скролл этапов ремонта с новыми визуалами, иконки материалов из UI kit и отключенное затемнение тарифных фото;
- `/v18/` - нативный закрепленный сценарий этапов на десктопе, последовательные карточки на мобильном, новый набор иконок и каталог материалов, CTA скрывается у футера;
- `/v19/` - спокойная мобильная история ремонта из трех статичных визуальных сцен: замер, работы и готовая квартира; осветлен градиент первого экрана;
- `/v20/` - бесконечная лента реальных фото с объектов ВЕРХ ремонта: свайп, стрелки и зацикливание в обе стороны; точные адреса не публикуются без согласия собственников;
- `/v21/` - упрощенная нативная галерея объектов без JS-зацикливания и маркированный демо-макет блока отзывов с AI-аватарами;
- `/` - навигация по версиям.

## Тарифы

- Стандарт: 20 000 ₽/м²
- Комфорт: 25 000 ₽/м²
- Люкс: от 29 000 ₽/м²

Финальная смета должна уточняться после замера и состава комплектации.
