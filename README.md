# ВЕРХ ремонт: версии прототипа сайта

Статичный сайт для GitHub Pages с версионированием макетов.

## Что внутри

- `/v1/` - первый кликабельный прототип;
- `/v2/` - версия ближе к сгенерированным desktop/mobile макетам;
- `/v3/` - версия с правками по Telegram: материалы по уровням ремонта, каталог обоев, контакты Telegram/MAX/VK/WhatsApp;
- `/v4/` - версия по полному разбору комментариев Юры: преимущества, быстрые контакты, подробные уровни ремонта и расчет от выбранного уровня;
- `/v5/` - сайт-квиз как интерактивная презентация: выборы сразу переводят к следующему шагу, заявка собирается в состоянии квиза, финал просит только телефон;
- `/v6/` - mobile-first квиз на одном экране: фиксированная высота без прыжков, фото-ответы, компактный прогресс и короткая заявка;
- `/v7/` - контентный сайт с квизом в iOS-style поп-апе: страница прогревает, квиз открывается поверх сайта;
- `/` - навигация по версиям.
