# ВЕРХ ремонт: прототип сайта

Статичный прототип для GitHub Pages.

## Что внутри

- первый экран с брендом и квизом;
- расчет стоимости по площади и тарифу;
- тарифы `Стандарт`, `Комфорт`, `Люкс`;
- блок материалов и каталогов;
- процесс ремонта;
- форма заявки без реальной отправки данных;
- визуальные концепты desktop и mobile.

## Тарифы

- Стандарт: 20 000 ₽/м²
- Комфорт: 25 000 ₽/м²
- Люкс: 29 000 ₽/м²

Финальная смета должна уточняться после замера и состава комплектации.
